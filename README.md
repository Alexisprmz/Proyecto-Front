# Frontend  (Angular)

Este proyecto corresponde a la interfaz de usuario . Está desarrollado en Angular (versión 19 o superior) utilizando componentes Standalone y establece comunicación con los microservicios responsables de autenticación (Node.js) y administración de propiedades (PHP).

## Características principales

- Arquitectura basada en componentes Standalone.  
- Gestión del estado mediante servicios con RxJS.  
- Estilos desarrollados con SCSS.  
  

## Requisitos previos

Antes de ejecutar el proyecto, es necesario disponer de:

- Node.js versión 18 o superior.  
- Angular CLI instalado de manera global.  

## Instalación

1. Clonar el repositorio correspondiente y acceder al directorio del proyecto.  
2. Instalar las dependencias necesarias mediante npm para habilitar Angular y las librerías adicionales utilizadas en la interfaz.
